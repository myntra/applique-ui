A. **Header** : contains the section title and is control for revealing the panel. 

B. **Optional icon (Chevron)** : indicates if the panel is open or closed. This icon can be replaced with any other icon.

C. **Main icon** : indicates what accordion says.

D. **Panel** : the section of content associated with an accordion header.
