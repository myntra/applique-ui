This feature-rich, interactive component lets users intuitively explore content. 
Unlike a traditional accordion, it's got a multi-level navigation that provides a linear path through the content. The header gives users a high-level overview of what to expect when they click on it. 
The first content section can then be expanded to reveal more information.

Accordion components are available in two styles: with an icon and without. 
Users can click the chevron to expand or collapse the component.

## States

With the Accordion component, you can use a single section to organize content in two different ways. This is ideal for the web where thereâ€™s limited screen space and you want to show a lot of information without cluttering up the screen.
This component will by default show in a collapsed state, and allow users to expand it to see more information.

In addition to the collapsed and expanded states, Accordion components also have interactive states for hover and disabled.

## Variants
