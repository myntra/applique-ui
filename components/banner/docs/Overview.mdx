## Dismissable Banner

```preview
function Example() {
  const [isVisible, setVisible] = React.useState(true);

  return isVisible &&
    <Banner type="warning" title={'CAUTION!'} onClose={() => setVisible(false)}>
      The image you are trying to upload has invalid format. It might not render well.
    </Banner>
}
```
