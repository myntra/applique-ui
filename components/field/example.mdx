## Examples

### Field Description

The `<Field>` component can display description for the input.

```jsx preview
const [value, setValue] = useState('');
const [v2, sv2] = useState('');

<div style={{display:"flex", flexWrap:"wrap"}}>
  <div style={{width:"400px", marginLeft:"16px"}}>
  <Field
    title="Name"
    description="Focused state with placeholder"
  >
    <InputText value={value} placeholder="Enter your name" onChange={(value) => setValue(value)} />
  </Field>
  </div>

  <div style={{width:"400px", marginLeft:"16px"}}>
  <Field
    title="Name"
    description="Enter your full name."
    error="Invalid Name"
  >
    <InputText value={value} onChange={(v) => setValue(v)} />
  </Field>
  </div>

    <div style={{width:"400px", marginLeft:"16px"}}>
      <Field
        title="Name"
        description="Enter your full name."
        success="Success message with hovered input text"
      >
        <InputText value={v2} onChange={(v) => sv2(v)} />
      </Field>
    </div>

    <div style={{width:"400px", marginLeft:"16px"}}>
      <Field
        title="Name"
        description="Enter your full name."
      >
        <InputText value="Filled Text Box" onChange={() => null} />
      </Field>
    </div>

    <div style={{width:"400px", marginLeft:"16px"}}>
      <Field
        title="Weight"
        description="Enter the Weight"
      >
        <InputText value="230" adornment={<p><b>/kg</b></p>} onChange={() => null} />
      </Field>
    </div>

    <div style={{width:"400px", marginLeft:"16px"}}>
      <Field
        title="Weight"
        description="Enter the Weight"
        error="weight must be less than 100"
      >
        <InputText value="230" adornment={<p><b>/kg</b></p>} onChange={() => null} />
      </Field>
    </div>

    <div style={{width:"400px", marginLeft:"16px"}}>
      <Field
        title="Weight"
        description="Enter the Weight"
        disabled
      >
        <InputText value="230" adornment={<p><b>/kg</b></p>} onChange={() => null} />
      </Field>
    </div>
</div>
```

## Input Select

```jsx preview
const [value, setValue] = useState();
const [options] = useState([
  { value: 'Red', label: 'Red' },
  { value: 'Green', label: 'Green' },
  { value: 'Blue', label: 'Blue' },
]);
const [weight, setweight] = useState('1');
const [optionsW] = useState([
  { value: '1', label: '1' },
  { value: '5', label: '5' },
  { value: '10', label: '10' },
]);

<>
<div style={{display:"flex", flexWrap:"wrap"}} >
  <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Color"
      description="Filled State"
    >
      <InputSelect
        options={options}
        value={value} onChange={setValue}
      />
    </Field>
  </div>
  <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Color"
      description="Choose your color"

        options={options}
        value={'Green'} onChange={setValue}
      />
    </Field>
  </div>
   <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Color"
      description="Choose your color"
      success="Everything is fine"
    >
      <InputSelect
        options={options}
        value={'Green'} onChange={setValue}
      />
    </Field>

  <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Color"
      description="A Disabled Input select"
      disabled
    >
      <InputSelect
        options={options}
        value={'Green'} onChange={setValue}
      />
    </Field>
  </div>

  <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Color"
      description="Empty selection "

      <InputSelect
        options={options}
        value={null} onChange={setValue}
      />
    </Field>
  </div>
    <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Weight"
      description="Adornment Text with focused"
    >
      <InputSelect
        options={optionsW}
        adornment={<p><b>/kg</b></p>}
        value={weight} onChange={setweight}
      />
    </Field>
  </div>

     <div style={{width:"400px", marginLeft:"16px"}}>
    <Field
      title="Weight"

    >
      <InputSelect
        options={optionsW}
        adornment={<p><b>/kg</b></p>}
        adornmentPosition="start"
        value={weight} onChange={setweight}
      />
    </Field>
  </div>
</div>
</>
```

### Required Field

The `<Field>` component displays a visual indication for required fields.

```jsx preview
<Field
  title="Name"
  required
>
  <InputText variant = "standard" value="" onChange={() => null} />
</Field>
```

### Error Message

The `<Field>` component can display error message in place of description.

```jsx preview
<Field
  title="Email"
  error="Invalid email address!"
>
  <InputText variant = "standard" value="" onChange={() => null} />
</Field>
```

We can run custom validations and provide error problematically. If `error` is present,
it will display the error, else it will display description.

```jsx preview
const [email, setEmail] = useState('');

<Field
  title="Email"
  description="Enter your email address."
  error={
    !email // if empty email field
    || /^[^@]+@myntra\.com$/.test(email) // or valid email value
      ? null // then no error
      : 'Invalid email address!'
  }
>
  <InputText
variant = "standard"     type="email"
    value={email}
    onChange={setEmail}
  />
</Field>
```
