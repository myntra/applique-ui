# Tooltip

<Documenter component={Tooltip}>

```jsx preview
<Tooltip renderContent={() => <div>Hello</div>}>
  <p>
    <Icon name={InfoCircleSolid} /> hover over me!
  </p>
</Tooltip>
```

</Documenter>

## Positioned Tooltips

```jsx preview

  display: 'flex',
  justifyContent: 'space-around'
}}>
  <Tooltip renderContent={() => <div>Hello</div>} position="up">
    <p>
      <Icon name={InfoCircleSolid} /> Top
    </p>
  </Tooltip>
  <Tooltip renderContent={() => <div>Hello</div>} position="down">
    <p>
      <Icon name={InfoCircleSolid} /> Bottom
    </p>
  </Tooltip>
  <Tooltip renderContent={() => <div>Hello</div>} position="left">
    <p>
      <Icon name={InfoCircleSolid} /> Left
    </p>
  </Tooltip>
  <Tooltip renderContent={() => <div>Hello</div>} position="right">
    <p>

```
