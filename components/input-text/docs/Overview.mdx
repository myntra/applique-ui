**Text input:** When the expected user input is a single line of text, as opposed to a paragraph.

**Text Area:** When the expected user input is more than one sentence.

The Input Fields component has a lot of variations we can use. It contains an input field, textarea, password fields, checkboxes, radio buttons, and everything else you need.

Fill, size, and width can all be mixed and matched to get the exact Input field needed.

## States

Input Fields have multiple bits and pieces, some of which are only visible depending on what state it is in (default, hover, active, filled, etc).

## Variants

### Single line text fields

An Input comes with a label display only one line of text

<Documenter component={props.components}>

```preview
const [value, setValue] = React.useState();

return (
    <Field title="Label">
        <InputText value={value} onChange={setValue} placeholder='Type / Scan'/>
    </Field>
)
```
</Documenter>

### Multi-line Text area

An Input comes with a label and grow to accommodate multiple lines of text

<Documenter component={props.components}>

```preview
const [value, setValue] = React.useState();

return (
    <Field title="Label">
        <InputTextArea value={value} placeholder='Type / Scan / Select' onChange={setValue} />
    </Field>
```

</Documenter>

### Messages

An Input should display descriptive validation information.

<Documenter component={props.components}>

```preview
const [value, setValue] = React.useState();
const [cost, setCost] = React.useState();

const InfoIcon = ({ children, title }) => (
    <svg
      xmlns="http://www.w3.org/2000/svg"
      style={{ color: '#4F47C8', width: '15px', height: '15px' }}
    >
        <title>{title}</title>
        {children}
    </svg>
)

return (
    <>
        <Field title="Label" description="Helper Text" required>
            <InputText value={value} onChange={setValue} placeholder='Type / Scan' adornment="/Kg" />
        </Field>
        <br/>
        <Field title="Label" success="Success">
            <InputText value={cost} onChange={setCost} placeholder='Cost' adornment="₹" adornmentPosition="start" />
        </Field>
        <br/>
        <Field title="Label" error="Error Message">
            <InputText value={cost} onChange={setCost} placeholder='Type / Scan' error />
        </Field>
        <br/>
        <Field title="Label" fieldInfo={<InfoIcon title="Useful extra information"><InfoCircleSolid/></InfoIcon>}>
            <InputText value={cost} onChange={setCost} placeholder='Type / Scan' />
        </Field>
    </>
)
```
</Documenter>
