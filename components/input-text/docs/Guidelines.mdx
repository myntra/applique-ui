A. **Label**: Indicates what sort of information the field requires and is left-aligned directly above the input area.

B. **Input Area**: This is where people enter text.

C. **Placeholder Text**: Lets people know how they should use the field, for example, “Type / Scan Item Code.”

D. **Main Icon**: Provide Visual hint as icon.

E. **Helper Text / Link / Message (Error and Success)**: Provides context about a field's input. and In filled State gives message.

F. **More info icon (optional)**: In certain cases where detailed help or context required this provide detailed info of an Input field
