# Variants
## Primary Button

Primary button calls attention to an action or highlights the most important call to action in a section. 

**Example : Save, Confirm, Done**

<Documenter >

```preview
return (
  <Button type="primary">Button Label</Button>
)

```

</Documenter>

## Secondary Button

Secondary buttons are used to highlight secondary actions. Remember to maintain visual hierarchy by balancing weights and background colours. 

<Documenter >

```preview
return (
  <Button type="secondary">Button Label</Button>
)

```

</Documenter>

## Tertiary Button

Tertiary buttons will indicate the least significant actions on the page. Pair them with primary and secondary buttons when there are multiple calls to action. 

<Documenter >

```preview
return (
  <Button type="tertiary">Button Label</Button>
)

```

</Documenter>

## Interactions
The <i>loading</i> prop can be used to show loading status.

```preview
function Example() {
  const [isLoading, setLoading] = React.useState(false)

  React.useEffect(() => {
    if (isLoading) setTimeout(setLoading, 3000, false)
  }, [isLoading])

  return (
    <Button onClick={() => setLoading(true)} loading={isLoading}>
      {'Start Now'}
    </Button>
  )
}
```
