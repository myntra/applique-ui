# Top Nav Bar
A Top Nav Bar is a graphical control element which contains drop-down menus or mega menus. The Top Nav Bar's purpose is to supply a common housing for window- or application-specific menus which provide access to such functions as opening files, interacting with an application, or displaying help documentation or manuals
