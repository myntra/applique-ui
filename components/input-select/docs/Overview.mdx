**Dropdown:** Allows the user to select one option from a list.

**Multi-select:** Allows the user to select multiple options from a list and filter.

The dropdown component has a lot of variations we can use. It contains a label and optional icons before or after the label. 

Fill, size, and width can all be mixed and matched to get the exact Dropdown needed.

## States
Dropdown have multiple bits and pieces, some of which are only visible depending on what state it is in (default, hover, active, inputted, etc).

## Variants

### Dropdown

By default, the dropdown displays Lable, placeholder text and chevron icon at the end.

<Documenter>

```preview
const [value, setValue] = React.useState();
const [options] = React.useState([
  { value: 'Red', label: 'Red' },
  { value: 'Green', label: 'Green' },
  { value: 'Blue', label: 'Blue' },
]);

return (
<>
  {'Label'}
  <InputSelect
    options={options}
    value={value} onChange={setValue}
  />
</>
)
```

</Documenter>

### Multi-select

By default, the Multi select displays Lable, placeholder text and chevron icon at the end. it can be used to select multiple options from a list or to filter information.

<Documenter>

```preview
const [value, setValue] = React.useState();
const [options] = React.useState([
  { value: 'Red', label: 'Red' },
  { value: 'Green', label: 'Green' },
  { value: 'Blue', label: 'Blue' },
]);

return (
<>
  {'Label'}
  <InputSelect
    multiple
    options={options}
    value={value} onChange={setValue}
  />
</>
)
```

</Documenter>
