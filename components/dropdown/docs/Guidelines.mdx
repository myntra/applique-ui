        <Anatomy />
    </Layout>

A. **Label** : Text that informs the user what to expect in the list of dropdown options.

B. **Selected Option Area**: Indicates what user have selected as an option.

C. **Placeholder Text**: Lets people know how they should use the field, for example, “Select an option.”

D. **Main Icon**: Provide Visual hint as icon.

E. **Helper Text / Link / Message (Error and Success)**: Assistive text to help the user choose the right selection or to show whats right or wrong happened.

F. **More Info Icon (optional)**: In certain cases where detailed help or context required this provide detailed info  of an Dropdown.

G. **Dropdown Icon**: Provide Visual hint of dropdown list.
